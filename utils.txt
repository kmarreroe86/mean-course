-Install Node.js

-Install angular-cli:
	npm install -g @angular/cli

-Visual Studio Code:
 Install Angular Essentials extension
 Install Material Icon Theme

-Create angular app:
	ng new "app-name"

-Add Angular Material
  ng add @angular/material

-Install express:
  npm install --save express

-Install nodemon
  npm install --save-dev nodemon
    (npm start "name_of_scripts" in packagejson)
   npm run  startnode

-Install body-parser (For parse requests body)
  npm install --save body-parser

-Install mongoose for MongoDB access:
  npm install --save mongoose

-Install cors Package for providing a Connect/Express middleware
npm install cors

-Install multer. Package for Express that allows to extract incoming files from request:
npm install --save multer


Debugging node:
  1. node --inspect server.js
  2. Open Google Chrome and open "about:inspect"
  3. Click "Open dedicated DevTools for Node"
